# Montella

Конвертер заказов доставки воды в Excel

## Возможности

- Импорт данных из таблицы заказов
- Автоматический подсчет по способам оплаты
- Генерация профессионального Excel отчета
- Сводка по наличным, бонусам и бесплатным заказам

## Запуск

### Docker
```bash
docker-compose up -d
```

### Локально
```bash
python3 -m http.server 8080
```

Открыть: http://localhost:8080
